# 仓储开票管理系统

## 项目简介

本项目是一个基于 Web 的开票管理系统。

系统主要用于中小型仓储销售场景，实现：

- 商品管理
- 客户管理
- 销售订单管理
- 开票打印
- 数据统计

系统采用前后端分离架构开发。

---

## 用户角色

### 管理员

负责系统整体管理。

### 销售人员

负责创建销售订单与开票。

### 仓库人员

负责库存管理与出库操作。
